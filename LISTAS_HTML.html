Listas en HTML5

En HTML5, podemos distinguir entre tres tipos de listas:
  • Listas Ordenadas: Son listas que se llaman OrderList (ol)
  • Listas Desordenadas: Son listas que se llaman UnOrderList (ul)
  • Listas de Definición: Son listas que se llaman Definition List (dl)

En las listas, podemos insertar cualquier tipo de elemento: texto, imagen, video, etc.
Podemos además, anidar listas de distintos tipos entre sí. Es decir, podemos mezclar
listas desordenadas con ordenadas, y viceversa.
Que la estructura de Listas Ordenadas / Desordenadas, se utilizan como base para realizar
los menús de nuestras páginas web (con CSS las ponemos bonitas)

Lista Ordenada: Ejemplo


La estructura de uan lista Ordenada, viene dada por una etiqueta OL de apertura y de cierre.
Además, internamente, se utiliza la etiqueta LI de apertura y cierra, para listar los elementos
de nuestra lista. Ya sea ordenada o desordenada.
Ejemplo de Lista Ordenada
  1. Windows 11
  2. Ubuntu
  3. Kali Linux
  4. Mac OS Yosemite

Lista Desordenada: Ejemplo


La estructura de la Lista Desordenada, viene dada por una etiqueta UL de apertura y de cierre
Además, internamente, se utiliza la etiqueta LI de apertura y cierra, para listar los elementos
de nuestra lista. Ya sea ordenada o desordenada.
Ejemplo de Lista Ordenada
  • Java
  • Python
  • JavaScript
  • PHP

Dar formato a Listas Ordenadas y Desordenadas

Para poder dar formato a las listas ordenadas y desordenadas, podemos utilizar
dos atributos que nos permitirán cambiar el inicio del comienzo de la
enumeración de las listas ordenadas así como el tipo,
y un atributo que nos permitirá cambiar el icono de nuestra lista ordenada.

Ejemplo de Lista Ordenada

  1. Podemos poner letras en Mayúsculas
  2. Podemos poner letras en Minúsculas
  3. Podemos poner números romanos en Mayúsculas
  4. Podemos poner números romanos en Minúsculas
  5. Por defecto, ponemos números "normales"
  6. Podemos indicar el orden en el que queremos comenzar

Ejemplo de Lista Desordenada

  • Podemos poner discos (viñeta redonda sin relleno) con disc
  • Podemos poner circulos sin relleno con el atributo circle
  • Podemos poner viñetas cuadradas con el atributo square

Listas de Definición

Las listas de Definición en HTML5, tienen una estructura totalmente distinta
a la estructura que presenta las listas ordenadas y desordenadas.
La estructura que presenta es la siguiente:
  • La etiqueta dl: Se conoce como Description List y se encarga
    de delimitar la lista completa. Tiene etiqueta de apertura y cierre
  • La etiqueta dt: Se conoce como Description Term.
    Delimitar los términos de la lista
  • La etiqueta dd: Se conoce como Definition Description.
    Delimita las definiciones

Ejemplo de Lista de Definición

DL
Crea la caja que contiene los items de la lista
DT
Item que contiene un término a definir
DD
Item que contiene una definición

Listas Anidadas

En HTML, podemos anidar listas tanto ordenadas como desordenadas unas dentro de otras.
Estos anidamientos, suelen establecerse con un máximo por lo general, de hasta un 3er nivel.
Ejemplo de listas anidada:
  • Primer curso DAM
    1. Entornos de Desarrollo
      1. Metodologías de Trabajo: Waterfull, Agile, Scrum, etc.
      2. Entornos de Desarrollo: Eclipse, IntelliJ, Netbeans, VS Code, etc.
      3. Refactorización con JUnit 5
      4. Git y Github
    1. Sistemas Informáticos
      1. Cambios de base: octal, binario, hexadecimal, decimal, etc.
      2. Hardware de PC (Sistemas Microinformáticos): FA, Placa Base, uProcesadores, HDD, T.Gráfica, etc.
      3. Sistemas Operativos Windows y Linux
      4. Virtualización (Máquinas Virtuales)
